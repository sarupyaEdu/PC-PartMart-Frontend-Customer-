# PC PartMart – Frontend (Customer)

PC PartMart Customer Frontend is a modern React + Vite + Tailwind CSS application built for a full-stack MERN e-commerce platform focused on PC components and accessories.

This frontend consumes a custom Node.js + Express REST API backend.

---

## 🚀 Features

- 🔐 JWT Authentication (Login / Register)
- 🛒 Cart Management
- ❤️ Wishlist System
- 📦 Bundle Products (Combo Deals)
- ⏳ Timed Offers with Countdown
- 💳 Razorpay Payment Integration
- 📦 Order History & Order Details
- 🔄 Returns & Replacements
- 🔍 Advanced Product Filtering
- 🏷️ Category & Brand Filtering
- ⭐ Product Reviews
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- React 19 (Vite)
- Tailwind CSS v4
- React Router
- Context API (Auth, Cart, Wishlist)
- Axios
- React Toastify
- Razorpay Checkout
- REST API Integration

---

## 🌐 Deployment

Designed for deployment on:

- Vercel (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
